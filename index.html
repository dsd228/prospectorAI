Cómo obtener los tokens
LinkedIn
1. linkedin.com/developers
2. Create App → asociar a tu página
3. Products → Share on LinkedIn
4. Auth → Client ID y Secret
5. Configurar en Ajustes → Redes
Instagram
1. developers.facebook.com
2. Create App → Business
3. Instagram Graph API
4. Necesitás cuenta Business/Creator
5. Configurar App ID y Secret
TikTok
1. developers.tiktok.com
2. Create App
3. Content Posting API (requiere aprobación 1-2 semanas)
4. Pegá el token manualmente
⚠ Configurá los Client ID / Secret en Ajustes → Redes Sociales antes de conectar.
